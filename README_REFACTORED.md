# Minecraft MOD Translator Tool (Refactored)

このツールは、MinecraftのMODファイル（.jar）から言語ファイルを抽出し、英語から日本語に翻訳するためのアプリケーションです。

## リファクタリングについて

このコードは元のMinecraft MOD Translator Toolを大幅にリファクタリングしたものです。主な改善点は以下の通りです：

### 1. モジュール構造の改善

コードを論理的な単位に分割し、より保守しやすい構造にしました：

```
src/
  ├── core/          # コア機能
  │   ├── file_manager.py    # ファイル操作
  │   ├── translator.py      # 翻訳機能
  │   └── resource_pack.py   # リソースパック生成
  ├── ui/            # UI関連
  │   ├── app.py             # メインアプリケーション
  │   ├── components.py      # 再利用可能なUIコンポーネント
  │   └── styles.py          # UIスタイルとテーマ
  └── utils/         # ユーティリティ
      ├── config.py          # 設定管理
      ├── exceptions.py      # カスタム例外
      └── logging.py         # ログ設定
```

### 2. 関心の分離

- ビジネスロジックとUIを分離
- 設定を集中管理するためのConfigクラスを導入
- 例外処理を改善し、カスタム例外クラスを導入

### 3. コードの再利用性向上

- 共通機能を再利用可能な関数やクラスに抽出
- UIコンポーネントを再利用可能な形で実装
- 依存性注入パターンを適用

### 4. エラー処理の改善

- より堅牢な例外処理
- 適切なエラーメッセージとログ記録
- ユーザーフレンドリーなエラー表示

### 5. コードの可読性向上

- 一貫した命名規則
- 適切なドキュメンテーション
- 型ヒントの追加

## 使い方

1. Minecraftのバージョンを選択します
2. 翻訳したいMODのjarファイルを選択します（または、modsフォルダのパスをクリップボードにコピーして「クリップボードからパスを取得」をクリックします）
3. 翻訳が完了すると、translate_rpフォルダに翻訳されたリソースパックが生成されます

## 必要なライブラリ

必要なライブラリは`requirements.txt`に記載されています。以下のコマンドでインストールできます：

```
pip install -r requirements.txt
```

## 実行方法

```
python main.py
```
